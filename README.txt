Contents
---------------

This is really just a dumping ground for some useful python scripts. It needs to be refactored and cleaned up
but this is a starting point

Classes
-------
BCU.py : Basic control of a BCU via snmp
SWI101.py : Basic control of a SWI101 via snmp
Kappa.py : Programming and setup of a Kappa camera
SNMPRequest.py : Backend of the snmp engine
bottle.py : Single page webframework http://bottlepy.org/docs/dev/index.html


Front End
----------
swiweb.py : Front end for the webinterface
views/ : The support templates for the swiweb interface
static/ : The static files like CSS and JS for the webinterface
